# SQL 문법 정리

## 가장 비싼 상품 구하기

### 사용한 SQL

```sql
SELECT MAX(PRICE) AS MAX_PRICE
FROM PRODUCT;
```

### 문제 이해하기

이 문제는 상품 정보를 담은 `PRODUCT` 테이블에서 가장 비싼 상품의 가격을 조회하는 문제입니다.

상품 가격은 `PRICE` 컬럼에 저장되어 있습니다.

여러 상품 가격 중 가장 큰 값을 찾아야 하므로 집계 함수인 `MAX()`를 사용합니다.

출력 컬럼명은 문제에서 요구하는 대로 `MAX_PRICE`로 지정합니다.

### 핵심 문법

#### 1. SELECT

```sql
SELECT MAX(PRICE) AS MAX_PRICE
```

`SELECT`는 조회할 값을 선택할 때 사용합니다.

이 문제에서는 모든 상품 정보를 출력하는 것이 아니라, 상품 가격 중 가장 큰 값 하나만 출력하면 됩니다.

따라서 `PRICE` 컬럼에 `MAX()` 함수를 적용합니다.

#### 2. MAX()

```sql
MAX(PRICE)
```

`MAX()`는 특정 컬럼에서 가장 큰 값을 구하는 집계 함수입니다.

`PRICE` 컬럼에 들어 있는 값들 중 가장 큰 가격을 찾기 위해 `MAX(PRICE)`를 사용합니다.

예를 들어 `PRICE` 값이 아래와 같다면,

| PRICE |
| --- |
| 10000 |
| 25000 |
| 18000 |

`MAX(PRICE)`의 결과는 가장 큰 값인 `25000`이 됩니다.

#### 3. AS

```sql
AS MAX_PRICE
```

`AS`는 조회 결과의 컬럼명에 별칭을 붙일 때 사용합니다.

`MAX(PRICE)`만 사용하면 결과 컬럼명이 데이터베이스에 따라 `MAX(PRICE)`처럼 표시될 수 있습니다.

문제에서 결과 컬럼명을 `MAX_PRICE`로 요구하므로 `AS MAX_PRICE`를 사용합니다.

#### 4. FROM

```sql
FROM PRODUCT
```

`FROM`은 데이터를 조회할 테이블을 지정할 때 사용합니다.

상품 정보는 `PRODUCT` 테이블에 들어 있으므로 해당 테이블에서 데이터를 조회합니다.

### 실행 흐름

1. `PRODUCT` 테이블에서 데이터를 조회합니다.
2. `PRICE` 컬럼의 값들을 확인합니다.
3. `MAX(PRICE)`로 가장 큰 가격을 구합니다.
4. 결과 컬럼명을 `MAX_PRICE`로 출력합니다.

### 최종 정리

이 문제의 핵심은 가장 큰 값을 구하는 집계 함수 `MAX()`를 사용하는 것입니다.

상품 가격 중 가장 비싼 가격만 필요하므로 `PRICE` 컬럼에 `MAX()`를 적용하고, 출력 컬럼명을 `MAX_PRICE`로 지정합니다.

```sql
SELECT MAX(PRICE) AS MAX_PRICE
FROM PRODUCT;
```
