# SQL 문법 정리

## 평균 일일 대여 요금 구하기

### 사용한 SQL

```sql
SELECT ROUND(AVG(DAILY_FEE), 0) AS AVERAGE_FEE
FROM CAR_RENTAL_COMPANY_CAR
WHERE CAR_TYPE = 'SUV'
GROUP BY CAR_TYPE;
```

### 문제 이해하기

이 문제는 자동차 대여 회사의 자동차 정보를 담은 `CAR_RENTAL_COMPANY_CAR` 테이블에서 자동차 종류가 `SUV`인 자동차들의 평균 일일 대여 요금을 구하는 문제입니다.

일일 대여 요금은 `DAILY_FEE` 컬럼에 저장되어 있습니다.

단, 모든 자동차의 평균을 구하는 것이 아니라 `CAR_TYPE`이 `'SUV'`인 자동차만 대상으로 해야 합니다.

또한 평균값은 소수 첫 번째 자리에서 반올림해야 하며, 출력 컬럼명은 `AVERAGE_FEE`로 지정해야 합니다.

### 핵심 문법

#### 1. SELECT

```sql
SELECT ROUND(AVG(DAILY_FEE), 0) AS AVERAGE_FEE
```

`SELECT`는 조회할 값을 선택할 때 사용합니다.

이 문제에서는 `DAILY_FEE`의 평균값을 구한 뒤 반올림해서 출력해야 합니다.

따라서 `AVG(DAILY_FEE)`로 평균을 구하고, `ROUND()`를 사용해 반올림합니다.

#### 2. AVG()

```sql
AVG(DAILY_FEE)
```

`AVG()`는 특정 컬럼의 평균값을 구하는 집계 함수입니다.

`DAILY_FEE` 컬럼에는 자동차의 일일 대여 요금이 저장되어 있으므로, `AVG(DAILY_FEE)`를 사용하면 대상 자동차들의 평균 일일 대여 요금을 구할 수 있습니다.

예를 들어 `DAILY_FEE` 값이 아래와 같다면,

| DAILY_FEE |
| --- |
| 10000 |
| 20000 |
| 30000 |

`AVG(DAILY_FEE)`의 결과는 `20000`입니다.

#### 3. ROUND()

```sql
ROUND(AVG(DAILY_FEE), 0)
```

`ROUND()`는 숫자를 반올림할 때 사용하는 함수입니다.

두 번째 인자에 `0`을 넣으면 소수점 아래 자리 없이 정수로 반올림합니다.

문제에서 평균 일일 대여 요금을 소수 첫 번째 자리에서 반올림하라고 했으므로 `ROUND(AVG(DAILY_FEE), 0)`을 사용합니다.

#### 4. AS

```sql
AS AVERAGE_FEE
```

`AS`는 출력 결과의 컬럼명에 별칭을 붙일 때 사용합니다.

문제에서 컬럼명을 `AVERAGE_FEE`로 출력하라고 했으므로 `AS AVERAGE_FEE`를 작성합니다.

#### 5. FROM

```sql
FROM CAR_RENTAL_COMPANY_CAR
```

`FROM`은 데이터를 조회할 테이블을 지정할 때 사용합니다.

자동차 대여 회사의 자동차 정보는 `CAR_RENTAL_COMPANY_CAR` 테이블에 들어 있으므로 해당 테이블에서 데이터를 조회합니다.

#### 6. WHERE

```sql
WHERE CAR_TYPE = 'SUV'
```

`WHERE`는 원하는 조건에 맞는 행만 조회할 때 사용합니다.

이 문제에서는 자동차 종류가 `SUV`인 자동차만 평균 계산 대상이 되어야 합니다.

따라서 `CAR_TYPE = 'SUV'` 조건을 사용합니다.

문자열 값인 `SUV`는 작은따옴표로 감싸야 합니다.

#### 7. GROUP BY

```sql
GROUP BY CAR_TYPE
```

`GROUP BY`는 특정 컬럼을 기준으로 데이터를 그룹화할 때 사용합니다.

이 문제에서는 `WHERE` 조건으로 이미 `SUV`만 남기기 때문에 결과 그룹은 `SUV` 하나입니다.

따라서 `GROUP BY CAR_TYPE`을 사용하면 `SUV` 자동차들만 묶어서 평균 일일 대여 요금을 계산할 수 있습니다.

### 실행 흐름

1. `CAR_RENTAL_COMPANY_CAR` 테이블에서 데이터를 조회합니다.
2. `CAR_TYPE`이 `'SUV'`인 자동차만 남깁니다.
3. 남은 자동차들을 `CAR_TYPE` 기준으로 그룹화합니다.
4. `DAILY_FEE`의 평균값을 구합니다.
5. 평균값을 소수 첫 번째 자리에서 반올림합니다.
6. 결과 컬럼명을 `AVERAGE_FEE`로 출력합니다.

### 최종 정리

이 문제의 핵심은 조건에 맞는 데이터만 필터링한 뒤 평균을 구하고 반올림하는 것입니다.

`WHERE`로 `SUV` 자동차만 선택하고, `AVG()`로 평균 일일 대여 요금을 구한 다음 `ROUND()`로 정수 형태로 반올림합니다.

```sql
SELECT ROUND(AVG(DAILY_FEE), 0) AS AVERAGE_FEE
FROM CAR_RENTAL_COMPANY_CAR
WHERE CAR_TYPE = 'SUV'
GROUP BY CAR_TYPE;
```
